# 42-Wolf3D
Ce miniproj consiste à créer graphiquement la représentation réaliste que l’on pourrait avoir à l’intérieur d’un labyrinthe. La représentation est réalisée en utilisant le principe du Ray-Casting.

	110/100 :)
	
## compil

	make

## run

	./wolf3d <map>

## keys

	MOVE : wasd
	JUMP : space
